## Setup

1. Install the dependencies

```bash
npm i -D @ncontiero/prettier-config prettier
```

2. Add the `prettier` key to your `package.json`

```json
{
  // ...
  "prettier": "@ncontiero/prettier-config"
  // ...
}
```
